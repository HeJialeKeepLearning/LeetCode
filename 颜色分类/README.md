# 题目
给定一个包含红色、白色和蓝色，一共 n 个元素的数组，原地对它们进行排序，使得相同颜色的元素相邻，并按照红色、白色、蓝色顺序排列。

此题中，我们使用整数 0、 1 和 2 分别表示红色、白色和蓝色。

注意:
不能使用代码库中的排序函数来解决这道题。

示例:

    输入: [2,0,2,1,1,0]
    输出: [0,0,1,1,2,2]

# 解题思路
空间换时间

v2（别人的思路）：荷兰旗问题，用2个指针分别指向0的右边界、2的左边界，然后一个指针从前向后遍历，如果指向0，则和右边界交换，指向2，则和左边界交换，指向1，则继续走，直到指针超过2的左边界为止。
