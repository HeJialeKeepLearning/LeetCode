## 题目
给出 n 代表生成括号的对数，请你写出一个函数，使其能够生成所有可能的并且有效的括号组合。

例如，给出 n = 3，生成结果为：

[
  "((()))",
  "(()())",
  "(())()",
  "()(())",
  "()()()"
]

## 解题思路
还是原先的想法。新的括号，是在原先的括号基础上，要么在里面添加，要么在外面添加。即：i=n时的括号，是'(' + p个括号 + ')' + q个括号，其中p + q = n - 1。

## 总结
之前的思路大体是对的，但是只考虑了p和q分别为0和n-1的情况，没有考虑p和q可以同时不为0，只要相加等于n-1即可。
