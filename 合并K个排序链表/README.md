# 题目
合并 k 个排序链表，返回合并后的排序链表。请分析和描述算法的复杂度。

示例:

    输入:
    [
      1->4->5,
      1->3->4,
      2->6
    ]
    输出: 1->1->2->3->4->4->5->6

# 解题思路
用k个指针分别指向k个链表的当前第一个结点，然后所有结点组成一个candidates数组，每次选candidates数组中最小的一个，然后pop出去之后，将选中的结点的next结点重新插入candidates数组。插入的时候用二分插入，这样时间复杂度是o(logk)

时间复杂度o(nlogk)，空间复杂度o(k)
