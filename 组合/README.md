# 题目
给定两个整数 n 和 k，返回 1 ... n 中所有可能的 k 个数的组合。

示例:

    输入: n = 4, k = 2
    输出:
    [
      [2,4],
      [3,4],
      [2,3],
      [1,2],
      [1,3],
      [1,4],
    ]

# 解题思路
[1~n]的k个整数的组合，是

    1 + [2~n]的k-1个整数组合
    2 + [3~n]的k-1个整数组合
    ...
    n - k - 1 + [n-k ~ n]的k-1个整数组合
想用动态规划做，但是没想出来。看别人的回溯法，思路类似，实现用了递归。
